LÉGENDE DES RISQUES PAR COMPOSANT
================================================================================

┌──────────────────────┬──────────────────────────────────────────────────────┐
│ Composant            │ Risque en cas de modification                        │
├──────────────────────┼──────────────────────────────────────────────────────┤
│ main.py (sanitize)   │ Incompatibilité avec scripts CI/CD existants         │
│ documents.py (repair)│ Faux positifs sur textes techniques (ex: "R0" → "R")│
│ documents.py (OCR)   │ pdf2image non installé → erreur silencieuse           │
│ vectorstore.py       │ NumpyVectorStore O(n) → lent pour >10k chunks       │
│ vectorstore.py (MMR) │ lambda_mult=0.5 non optimisé pour corpus gazier     │
│ llm.py (retry)       │ max_retries=3 insuffisant pour APIs instables        │
│ config.py (prompt)   │ Refusal instruction → échec systématique génération │
│ pipeline.py (valid.) │ Clé JSON manquante → crash Unity VR                 │
│ prompts.py (schema)  │ JSON_SCHEMA_V2 désynchronisé → Unity plante         │
│ json_converter.py    │ Markdown mal nettoyé → JSON invalide                 │
└──────────────────────┴──────────────────────────────────────────────────────┘
